HOW TO USE

  • Step 1: Select the shape from the first dropdown menu.
  • Step 2: Select the style/algorithm you want the maze to get generated in.
  • Step 3: Type the height and the width in their respective places.
  • Step 4: Generate your maze.

LIMITATIONS

  • SQUARE: You can create the size up to 140x140.
  • TRIANGLE: You can create the size up to 90x90.
  • HEXAGONAL: You can create the size up to 90x90.
  • DIAMOND: You can create the size up to 100x100.
  • SQUARETRIANGLE: You can create the size up to 70x70.
  • SNUBSQUARE: You can create the size up to 110x110.
  • SNUBSQUARE2: You can create the size up to 80x80.
  • CAIRO: You can create the size up to 140x140.
  • CAIRO: You can create the size up to 60x60.
  • PENROSE-SUN: You can create the size up to 7x7.
  • PENROSE-STAR: You can create the size up to 7x7.
Larger maze sizes might generate slower.
